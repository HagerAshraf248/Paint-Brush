# Paint-Brush


## Abstract

Paint Brush is a Java Swing (JFrame-based) drawing application that allows users to draw basic shapes using different colors and line styles. The program also supports free-hand drawing, erasing, clearing the canvas, and undoing previous actions.

## Features

### Colors

Users can choose a drawing color before creating shapes.
Available colors: Red, Green, Blue.

### Shapes

Users can draw shapes by dragging the mouse from a start point to an end point. The shape is previewed during dragging and finalized on mouse release.
Supported shapes:

* Rectangle
* Oval
* Line

### Free Hand

Allows users to draw freely by dragging the mouse to create continuous lines.

### Eraser

Enables users to erase parts of drawings by dragging over them.

### Undo

Allows users to undo the most recent drawing actions.

### Clear All

Removes all drawings from the canvas.

### Dotted Option

When selected, shapes are drawn with dotted lines. Otherwise, lines are solid.

### Filled Option

When selected, Rectangle and Oval shapes are filled with the chosen color. Otherwise, they are outlines only.

## Technologies Used

Java, Java Swing, Graphics2D, Event Handling
